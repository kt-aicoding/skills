# KT AI Coding Skills

面向 AI 编程 Agent 的可复用技能集合。

## 边界

这个仓库收纳 Codex、Claude Code、Cursor、Windsurf 等 AI 编程环境中的技能、指令和工作流 playbook。

## 目录规划

```text
skills/
  codex/
  claude-code/
  shared/
```

## 规范

- 每个 skill 聚焦一个可重复执行的工作流。
- 跨客户端通用的内容优先放到 `shared/`。
- 客户端特有行为放到对应客户端目录。
- 不收纳应用 demo 或无关 prompt；这里专注 AI 编程工作流。
