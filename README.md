# KT AI Coding Skills

Reusable skills for AI coding agents.

中文说明见 [README.zh-CN.md](README.zh-CN.md)。

## Scope

This repository collects skills, instructions, and workflow playbooks for Codex, Claude Code, Cursor, Windsurf, and other agentic coding environments.

## Layout

```text
skills/
  codex/
  claude-code/
  shared/
```

## Guidelines

- Keep each skill focused on one repeatable workflow.
- Prefer portable instructions when a skill works across multiple AI coding clients.
- Put client-specific behavior under a client-specific directory.
- Avoid app demos and unrelated prompts; this repository is for AI coding workflows.
